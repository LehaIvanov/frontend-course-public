# EVENT-CALENDAR

Учебный проект для изучения React и React Hooks.

## Описание работы приложения

На экране пользователя должно отображаться 5 карточек - 5 дел пользователя, 4 столбца - 4 недели. Карточки расположеных в этих столбцах, и пользователь имеет возможность перемещать карточки как внутри одного столбца, так и между столбцами. Перемещение карточек осуществляется с помощью мыши.

То есть вам нужно реализовать функционал Drag and Drop.

### Требования

- При перетаскивании должно подсвечиваться потенциально новое место карточки синим цветом, и предыдущее место карточки серым цветом. Таким образом пользователь может видеть откуда он перетаскивает карточку и куда.

- Нельзя использовать сторонние библиотеки. Для релизации необходимо использовать стандартные средства React. Можно использовать возможности HTML 5 Drag and Drop (события drag, dragstart, dragend, и т.д.), но я советую этого не делать. Лучше использовать стандартные события mousedown, mousemove, mouseup.

- В DOM элемента руками не лезем все реализуем через React.

### Советы реализации

Можно использовать хук useReducer для переключения между состояиями dragstart, dragend, drag.

Можно зашить размеры карточек, Responsible Layout можно не реализовывать.

Можно не заморачиваться со скроллом. Поддержка скролла страницы в функционале Drag And Drop добавляет некоторых проблем, хоть и небольших. Можно скролл не поддерживать, представим, что все наши карточки помещаются на странице.
